## 📝 Question Cards

Each question is presented as an interactive card:

### Card Elements

```
┌─────────────────────────────────┐
│ Q1    Workspace Architecture    │  ← Header with question number
├─────────────────────────────────┤
│                                 │
│ Objective & Key Concepts        │  ← Description & summary

│                                 │
│ Relevant Topics: X, Y, Z        │
│                                 │
├─────────────────────────────────┤
│                                 │
│ [View Details] [Solution] ←     │  ← Action Buttons
│  or Download                    │
│                                 │
└─────────────────────────────────┘
```

### Card Color Coding

- **🟨 Yellow Border (Left)**: Theory questions (Section A & B)
- **🟩 Green Border (Left)**: Practical questions (Section C)
- **🟦 Blue Border (Left)**: Interpretation & Resources

### Interactive Features

**1. Hover Effect:**
   - Card lifts up slightly
   - Shadow becomes more prominent
   - Smooth animation

**2. View Details Button:**
   - Click to open a modal popup
   - Shows full explanation
   - Includes pseudocode, examples, and concepts
   - Click the X to close

**3. Solution/Download Button:**
   - Opens the actual solution file
   - Opens in new tab for easy switching
   - Can be HTML, Python, or other formats

## 📓 Section C: Practical Notebook

### What's Included
- **Jupyter Notebook:** Complete data analytics pipeline
- **Dataset CSV:** 8 student records with metrics
- **Explanations:** Step-by-step walkthrough

### How to Navigate
1. Scroll to "📓 SECTION C" or click "Section C" in menu
2. View two cards: Notebook and Dataset
3. **"View Explanation"** button → See full walkthrough
4. **"Notebook"** button → Opens interactive Jupyter file
5. **"📥 Download CSV"** button → Gets dataset

### Using the Jupyter Notebook

**Option 1: Local Jupyter**
```bash
# Install Jupyter if needed
pip install jupyter

# Navigate to the notebook location
cd Section_C/Practical_Notebook

# Start Jupyter
jupyter notebook

# Click on Section_C_Notebook.ipynb to open
```

**Option 2: Online (Google Colab)**
1. Go to: https://colab.research.google.com
2. File → Upload notebook
3. Select `Section_C_Notebook.ipynb`
4. Run all cells

**Option 3: VS Code**
1. Open VS Code
2. Install Jupyter extension if needed
3. Open the `.ipynb` file
4. Run cells one by one or all at once

### Dataset Usage
- Download the CSV file
- Open in Excel, spreadsheet app, or text editor
- Contains 8 student records with columns:
  - Name, Department, Attendance, Diary Entries, Tasks, Assignment Score, Lab Score

## 🐛 Troubleshooting

### Modal won't close
- Click the X button again
- Click outside the modal box
- Press Escape key
- Refresh the page

### Links not working
- Check internet connection
- Ensure GitHub Pages is deployed
- Try refreshing the page
- Check if files exist in repository

### Dashboard looks strange
- Try different browser
- Clear browser cache (Ctrl+Shift+Delete)
- Try incognito/private window
- Check screen resolution

### Can't download files
- Click and hold to see options
- Right-click → Save As
- Try different browser
- Check file permissions
