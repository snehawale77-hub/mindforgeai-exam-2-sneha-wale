# MindforgeAI Module 2 Assessment: Student Analytics Pipeline

## 🎯 Objective
The objective of this project is to build an automated data analytics pipeline that evaluates student academic signals (attendance, diary entries, task completion, and lab scores). The system calculates a weighted readiness score, classifies students into performance bands, and extracts actionable insights for faculty intervention.

## 🛠️ Tools Used
* **Language:** Python 3
* **Libraries:** Pandas (for data manipulation and analysis)
* **Environment:** Jupyter Notebook / VS Code
* **Version Control:** Git & GitHub

## 🚀 How to Run the Project
1. Clone this repository to your local machine.
2. Ensure you have Python and Pandas installed (`pip install pandas`).
3. Navigate to the `notebooks/` directory.
4. Open `Section_C_Notebook.ipynb` using Jupyter or VS Code.
5. Click **"Run All"** to execute the pipeline. The notebook will automatically pull the raw data from the `data/` folder.
