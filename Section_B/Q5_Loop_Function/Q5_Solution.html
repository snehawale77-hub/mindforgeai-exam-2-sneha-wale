🛠️ Custom Readiness Score Function

The Logic Weightage:
To calculate a score out of 100, we apply weights:
• Attendance: 40% of final score (Max 100%)
• Diary: 30% of final score (Max 15 entries)
• Tasks: 30% of final score (Max 10 tasks)
Slide the inputs below to see how the Python function calculates the total and categorizes the student!

Student Inputs

def readiness_score(att_pct, diary, tasks):
# Calculate weighted parts
att_score = att_pct * 0.40
diary_score = (diary / 15) * 30
task_score = (tasks / 10) * 30

total_score = att_score + diary_score + task_score

# Apply Control Flow
if total_score >= 80:
band = "Ready for Deployment"
elif total_score >= 50:
band = "Needs Improvement"
else:
band = "Critical Support Needed"

return total_score, band
