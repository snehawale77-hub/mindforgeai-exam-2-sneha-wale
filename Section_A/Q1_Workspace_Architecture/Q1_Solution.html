This hybrid workflow ensures that local processing, cloud execution, version control, and manual theoretical work are properly isolated and managed using the following tools:

Platform / Tool Exam Responsibility Core Task Execution
🖥️ VS Code Local Workspace Management Initializing the local folder structure, writing clean markdown explanations, and formatting `.py` scripts.
