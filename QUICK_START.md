```bash
git push origin main
```

### Minute 4️⃣ - Enable GitHub Pages
1. Go to your GitHub repository
2. Click **Settings** (top right)
3. Click **Pages** (left sidebar)
4. Set Source: **main** branch → **/ (root)**
5. Click **Save**

## 📞 Troubleshooting

### Dashboard won't load locally
```
→ Ensure index.html exists in root folder
→ Try opening with different browser
→ Check for file corruption
```

### GitHub Pages not updating
```
→ Verify files are pushed (git push succeeded)
→ Check Settings → Pages shows "Active"
→ Wait 5-10 minutes for deployment
→ Do hard refresh: Ctrl+Shift+Delete
```

### Links broken in dashboard
```
→ Verify all files are in repository
→ Check file paths in index.html
→ Re-commit and push changes
```
