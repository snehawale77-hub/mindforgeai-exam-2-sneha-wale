## 📞 Support & Troubleshooting

### Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Dashboard won't load | Check GitHub Pages settings, wait 5 min |
| Links broken | Verify file paths, update links in index.html |
| Styling looks wrong | Clear browser cache (Ctrl+Shift+Del) |
| Mobile looks bad | Test in DevTools, check media queries |
| Files not found | Verify files committed to Git, check paths |
| GitHub Pages not updating | Push to main branch, wait 2-5 min |

### Quick Reference

**Local Testing:**
```bash
# Open index.html directly in browser
```
